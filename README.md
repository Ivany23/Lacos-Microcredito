# Lacos-Microcredito

API NestJS para gestão de clientes e empréstimos.

## Tecnologias

- NestJS
- TypeORM
- PostgreSQL
- Swagger UI (Documentação)

## Instalação

```bash
npm install
```

## Execução

```bash
# desenvolvimento
npm run start:dev

# produção
npm run start:prod
```

## Documentação API

A documentação Swagger estará disponível em `/api` após iniciar o servidor.
